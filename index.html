Form Code

User-based algorithmic compositions (interactive art & gallery)

In this project I experimented with algorithmic composition using the web's Canvas API (for creating interactive and generative 2D drawings) via the p5.js library. The goal was to create an algorithmic (visual) composition where the code produces different artistic outputs depending on user input.

The user has multiple modes and configurations to manipulate including a free mouse moving option. Lastly, once the user creates an art piece, they can view the gallery which consists of all pieces made by other users. This project was made using JavaScript, HTML, CSS, and Firebase.

Internet Age

Statement of life in the age of internet (w/VR)

This project is a statement about life in the age of the internet. It features commentary on surveillance, automation, and social media. It includes elements of WebVR using the A-Frame framework.

The user has multiple courses of action on the introduction screen, and can roam around the virtual worlds (using mouse and WASD/arrows) in the VR section. After the tour, the user can view each world separately in the gallery section. This project was made using HTML, CSS, and JavaScript.

To Space and Back

A fraction learning space / rocket game

To Space and Back is a fraction learning game for 3rd-5th graders. The game is designed to introduce the concepts of fractions through a graphical representation of fractions as slices of circles and rectangles. The game is made to accompany the initial steps in the fractions learning trajectory for the Common Core curriculum for our target age group.

This project was made with an emphasis on accessibility for students of various backgrounds through the implementation of customization and references. Players can choose between different background music options, text-to-speech, and various languages. We also took into account users with various disabilities, including ESL, auditory impairments, visual impairments, attention deficit. This game was made in collaboration with Zack Crenshaw and is made/run using Java.

Friendship Quest

A 2D RPG fantasy friendship simulator

Dreamdrift: Friendship Quest is an action/adventure RPG about finding friendships within a capitalist world. This demo explores the dialogue and choices throughout part of our main storyline and reveals the backgrounds of key characters. The player’s objective is to travel toward the “Dreamdrift Isles”, an unreachable fantasy land that represents the anxiety and uncertainty shared by our generation in the modern world.

Choice is the main mechanic in the game. Throughout the game, players encounter NPCs whose dialogue encourages them to make choices within the gameworld. These choices affect the gameplay through a branching narrative. Players also encounter battles, a gold system, and cutscenes. This project was made in a group of 5 using RPG Maker MV, and is available for Mac and Windows.
